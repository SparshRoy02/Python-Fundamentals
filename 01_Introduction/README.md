## 01 : Introduction To Python
- Python is a high-level, interpreted programming language known for its readability and simplicity.
- It is versatile and widely used in various domains, including web development, data analysis, artificial intelligence, and more.
- Python code is executed line by line, making it an excellent choice for beginners.

### What is Python ?

Python is a **dynamically typed**, **general-purpose programming language** that supports:
- Object-Oriented Programming
- Functional Programming
- Python is an **interpreted** and **high-level language**, which makes it beginner-friendly and easy to debug.
- Created by **Guido van Rossum** in **1989**

### Features of Python

- Simple and easy to understand
- Interpreted and platform-independent
- Open-source
- Extensive library support:
  - NumPy
  - Pandas
  - TensorFlow
  - Selenium
  - OpenCV
- Supports integration with other languages

### What is Python Used For?
- Data Visualization :
   - Creating plots, charts, and graphical representations of data
- Data Analytics :
   - Analyzing raw data to extract insights and identify trends
- Artificial Intelligence & Machine Learning :
   - Simulating human intelligence
- Web Development :
   - Building dynamic and scalable web applications
- Database Management :
  -  Connecting, managing and manipulating databases
- Business & Accounting Applications :
   - Performing complex mathematical operations

### What is Syntax?
In the simplest words, Syntax is the arrangement of words and phrases to create well-formed sentences in a language. In the case of a computer language, the syntax is the structural arrangement of comments, variables, numbers, operators, statements, loops, functions, classes, objects, etc., which helps us understand the meaning or semantics of a computer language.
